## Get-started
1. 安装好go环境
2. go run main.go
3. 浏览器访问localhost:8081

## 目录说明
- 除了main.go以外，go项目的主要代码在src目录下
- dist目录存放的是vue项目打包后的文件
- vueblog存放的是vue项目原始文件
- uploads存放的是项目文件上传api的target directory
